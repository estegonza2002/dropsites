How it works

From file to live link.
Three steps.

No configuration. No servers. No deployment pipelines. Just drop and share.

01
Drop your file
Drag a single HTML file, a ZIP archive, a React build, a PDF, or any static format onto the upload zone. Or use the API, CLI, or Claude MCP connector.
02
Get a permanent link
DropSites processes your file and returns a stable, shareable URL in under 60 seconds. Custom slugs, namespaces, and custom domains are supported.
03
Share anything
Copy the link, embed it, send a QR code, generate per-recipient tracking tokens, or set a password. Your file is served globally via Cloudflare CDN.

Features

Everything a static file deserves

Every feature ships in Phase 1. Nothing locked behind enterprise gates unless noted.

Any static format
HTML, ZIP, React builds, JS, PDF, WASM, images. If a browser can render it, DropSites can serve it.
Instant overwrite
Re-upload to any existing slug. The URL never changes. Anyone with the link sees the updated version immediately.
Password protection
Lock any deployment behind a password. Brute-force rate limiting built in. Toggle on or off from the dashboard row.
Per-recipient tracking
Generate named access tokens for each recipient. Know exactly when Alice opened your proposal — not just total view counts.
Workspace model
Deployments belong to workspaces — not individuals. Shared namespaces, role-based access, team billing. Members come and go; links stay.
In-browser editor
Edit any file in your deployment directly in the browser. Save and publish in one click — change is live immediately, no re-upload required.
Link expiry
Set an expiry date on any deployment. Expired links show a branded page. Reactivate in one click — files are never deleted on expiry.
Zero egress costs
Built on Cloudflare R2 and CDN. No egress fees at any scale. 15x cheaper than S3 at 100k users. Cost advantage is structural, not a promo.
Full REST API + CLI
Headless publishing from any script, CI pipeline, or AI agent. Official JS and Python SDKs. GitHub Actions action. Webhooks with HMAC signing.
